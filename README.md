

## Estructura del Archivo

Tras su instalación se desplegará los siguientes archivos así:

```
my-app/
  README.md
  node_modules/
  components/
    Banner.js
    Filters.js
    Footer.js
    Lastest.js
    MovieInfo.js
    Preview.js
    Search.js
  multimedia/
  package.json
  public/
    index.html
    favicon.ico
  pages/
    Home.js
    Movie.js
  src/
    App.css
    App.js
    App.test.js
    index.css
    index.js
    logo.svg
```
